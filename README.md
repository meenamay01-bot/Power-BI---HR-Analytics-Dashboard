# 📊 HR Analytics Dashboard – Power BI

## 📌 Project Overview
This dashboard analyzes employee attrition data for a company of 1,470 employees.
The goal is to help HR teams identify attrition patterns and take data-driven decisions
to improve employee retention.

## 🔢 Key Metrics
| Metric | Value |
|---|---|
| Total Employees | 1,470 |
| Attrition Rate | 16.12% |
| Average Age | 36.92 years |
| Avg Monthly Income | ₹6,503 |

## 📈 Insights from the Dashboard
- **Laboratory Technicians** have the highest attrition count by job role
- Employees in the **Under 30 and 30–39** age groups leave the most
- **Research & Development** department has the highest attrition by department
- **Sales department** has the highest average monthly income
- Workforce is **60% Male and 40% Female**

## 🛠️ Tools Used
- Power BI Desktop
- Microsoft Excel (data source)
- DAX (for calculated measures)
